
Within a virtual Ubuntu environment:

```
sudo apt install libgirepository1.0-dev gcc libcairo2-dev pkg-config python3-dev gir1.2-gtk-3.0 texlive-xetex python3-setuptools gir1.2-poppler-0.18 gir1.2-gtksource-3.0 python3-numpy
git clone --depth=1 https://github.com/sillsdev/ptx2pdf.git
cd ptx2pdf
python3 -m pip install -r requirements.txt
```
